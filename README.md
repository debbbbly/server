# Debbly Backend

This is the Kotlin/Spring Boot backend for Debbly, a live video debating platform.

## ✅ Features
- User authentication (AWS Cognito)
- Debate matching system
- Live debate room management
- ...

## 🔧 Tech Stack
- Kotlin + Spring Boot
- AWS Cognito (auth)
- PostgreSQL (main DB)
- Redis (matchmaking queue)
