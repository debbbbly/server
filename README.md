# Debbly Backend

This is the Kotlin/Spring Boot backend for Debbly, a live video debating platform.

## ✅ Features
- User authentication (Self hosted Supabase Auth (GoTrue))
- Debate matching system
- Live debate room management
- ...

## 🔧 Tech Stack
- Kotlin + Spring Boot
- Supabase Auth (GoTrue, self hosted) (auth)
- PostgreSQL (main DB)
- Redis (matchmaking queue)
